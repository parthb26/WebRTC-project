# WebRTC Communication Project
A lightweight, scalable WebRTC-based project enabling real-time peer-to-peer (P2P) communication using Node.js for signaling and TypeScript for maintainable, scalable code.

## Features

- **Real-Time Communication**: Peer-to-peer video, audio, and data transfer using WebRTC.
- **Efficient Signaling**: Node.js server for signaling with secure WebSocket communication.
- **Modern Frontend**: React.js with TypeScript for building an intuitive user interface.

## Tech Stack

- **Frontend**: React.js with TypeScript
- **Backend**: Node.js (Express)
- **WebRTC**: Peer-to-peer connection
- **Other Tools**: WebSocket for signaling

## Installation and Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/webrtc-project.git
   cd webrtc-project
2. Install dependencies:
 # Backend  
cd Backend  
npm install  

# Frontend  
cd ../Frontend  
npm install  

3. Start the development servers:
# Backend  
cd Backend  
npm start  

# Frontend  
cd ../Frontend  
npm start  

4. Open your browser and go to http://localhost:3000.

# Usage
Connect users over peer-to-peer WebRTC channels with seamless signaling for real-time communication. Extend features to include chat, file sharing, or other custom interactions.
